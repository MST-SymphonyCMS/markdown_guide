<strong>Headers:</strong> Precede with a # for each level desired as in: ### This is an h3 header<br />
<strong>Paragraphs:</strong> Seperate with a blank line<br />
<strong>Blockquotes:</strong> Precede with a >  (two >> will produce a quote within a quote)<br />
<strong>Unorderd lists:</strong> Precede list items with a *, + or -<br />
<strong>Ordered lists:</strong> Precede list items with a number followed by a period<br />
<strong>Emphasis:</strong> *italics*  **bold**  or  _italics_  __bold__<br />
<strong>Links:</strong> [link text](http://some-site.com "optional title")<br />
<strong>Automatic links:</strong> Enclose a URL in angle brackets to produce a link using the URL itself as the link text<br />
<strong>Images:</strong> ![alternate text](/path/to/img.jpg "optional title")<br />
<strong>Code blocks:</strong> Indent each line of the block by at least 4 spaces or one tab<br />
<strong>Inline code:</strong> Wrap the code with backtick characters<br />
<strong>Horizontal rules:</strong> Place 3 or more asterisks, hyphens or underscores on a line by itself<br />
<strong>Line breaks:</strong> To force a line break, end the line with 2 spaces and then start a new line<br />
<strong>Backslash escapes:</strong> To force a literal display of special markdown characters, precede with a backslash<br />
